Plugins

silk-sprite.css

  • With the silk-sprite.css plugin you can make your unordered list looking prettier.

Below is an <ul class="ui-silk ui-silk-star">:

  • Lorem ipsum dolor sit amet.
  • Consectetur adipisicing elit.
  • Sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.
  • Nunc tincidunt vehicula pede.
  • Below is a nested <ul class="ui-silk ui-silk-lightning">:
    • Lorem ipsum dolor sit amet.
    • Consectetur adipisicing elit.
    • Sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.
    • Et dolore magna aliqua.
    • Nunc tincidunt vehicula pede.

To show difference icon for difference list item, use <ul class="ui-silk"> then compose <li> like the following example:

  • <li class="ui-silk-add"> List item #1.
  • <li class="ui-silk-delete"> List item #2.
  • <li class="ui-silk-accept"> List item #3.
  • The silk-sprite.css plugin also enables you to use Silk icons anywhere you like in your page.

Move your mouse pointer over the class name beside an icon to read usage tip.
