About

F6 Liquidus CSS framework is the combination of several well-known CSS frameworks. The 960 Grid System, the Blueprint and the Emastic CSS Framework to name a few. So, why need another CSS framework? F6 Liquidus CSS framework is created to inherit the pros of existing frameworks while trying to limit their cons to minimal. Besides that, F6 Liquidus CSS framework also contains many build-from-scratch proprietary styles.
